# SGelectric

Graphing SG USEP, Brent Crude Price and Energy Demand over time

1. Setup

pip install pandas, matplotlib

2. Data Sources:

a) USEP Data - National Electricity Market of Singapore (https://www.nems.emcsg.com/nems-prices). Download annual data in zip format.

b) Brent Crude Prices - U.S. Energy Information Administration via FRED® (https://fred.stlouisfed.org/series/DCOILBRENTEU). Download same period in excel file
